# Modern Pascal Learning Journey

This repository tracks my progress through the **"Modern Pascal"** book series by **Corin Hale**. It serves as a personal development log and educational resource for others interested in Free Pascal and Lazarus.

## 📚 Book Series

1. **Modern Pascal Fundamentals** (Current)
2. **Structured Pascal Techniques**
3. **Mastering Pascal Data Structures**
4. **Object-Oriented Pascal**
5. **Advanced Pascal Architecture**
6. **Practical Pascal Projects**

## 📁 Project Structure

- `ModernPascalFundamentals/`: Projects and exercises from the first book.
- `[BookName]/`: Projects for subsequent books in the series.
